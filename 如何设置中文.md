# 如何设置中文界面

由于直接修改默认语言会导致编译错误，我们采用更稳定的方式来实现中文界面。

## 方法一：首次启动时安装（推荐）

1. 首次启动 VS Code
2. 右下角会提示安装中文语言包
3. 点击 "Install and Restart" 安装并重启
4. 重启后即为中文界面

## 方法二：手动安装语言包

1. 打开 VS Code
2. 按 `Ctrl+Shift+X` 打开扩展面板
3. 搜索 "Chinese (Simplified)"
4. 安装 "Chinese (Simplified) Language Pack for Visual Studio Code"
5. 按 `Ctrl+Shift+P` 打开命令面板
6. 输入 "Configure Display Language"
7. 选择 "zh-cn"
8. 重启 VS Code

## 方法三：修改配置文件

1. 按 `Ctrl+Shift+P` 打开命令面板
2. 输入 "Configure Display Language"
3. 点击 "Install additional languages..."
4. 安装中文语言包
5. 选择 "zh-cn"
6. 重启 VS Code

## 方法四：使用命令行

```bash
# 安装中文语言包扩展
code --install-extension MS-CEINTL.vscode-language-pack-zh-hans

# 设置语言为中文
code --locale=zh-cn
```

## 永久设置中文

编辑 `~/.vscode-oss/argv.json` 或 `%APPDATA%\Code - OSS\argv.json`：

```json
{
  "locale": "zh-cn"
}
```

## 为什么不默认中文？

直接修改源码的默认语言会导致：
- ❌ Linux 和 macOS 编译失败
- ❌ 语言包加载问题
- ❌ 与上游代码冲突

使用语言包的方式：
- ✅ 编译稳定
- ✅ 官方支持
- ✅ 易于维护
- ✅ 可以随时切换语言
