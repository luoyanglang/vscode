# 如何运行编译后的 VS Code

## 问题

编译后的代码（`out/` 目录）不能直接运行，本地环境配置也比较复杂。

## 解决方案

### 方案1：下载官方 VS Code（推荐）

1. 访问：https://code.visualstudio.com/
2. 下载 Windows 版本
3. 安装后直接使用

**优点：** 简单快速，开箱即用

### 方案2：使用便携版 Electron

1. 下载 Electron：https://github.com/electron/electron/releases
2. 解压到某个目录，比如 `D:\electron\`
3. 把编译好的 `out/` 目录复制到 `D:\electron\resources\app\`
4. 运行 `D:\electron\electron.exe`

**优点：** 可以使用自己编译的代码

### 方案3：配置完整的开发环境

需要安装：
- Visual Studio 2022（包含 C++ 构建工具和 Spectre 缓解库）
- Python 3.x
- Node.js
- 所有依赖包

然后运行：
```bash
npm install
npm run compile
.\scripts\code.bat
```

**缺点：** 配置复杂，耗时较长

## 建议

如果只是想使用 VS Code，直接下载官方版本。

如果想开发和调试 VS Code，需要配置完整的开发环境。

## GitHub Actions 编译

已经配置好了 GitHub Actions 自动编译，每次推送代码都会自动编译。

编译产物可以在 Actions 页面下载：
https://github.com/luoyanglang/vscode/actions/workflows/build-vscode.yml

但下载的只是编译后的代码，仍然需要配合 Electron 运行。
