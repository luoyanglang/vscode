# 小玉智能助手

集成小玉智能系统的 VS Code AI 编程助手，让 AI 成为你的编程伙伴。

## ✨ 功能特性

### 基础功能
- 💬 **AI 聊天面板** - 侧边栏实时对话，支持会话管理
- 🔍 **代码解释** - 选中代码右键解释功能和实现原理
- ♻️ **代码重构** - AI 辅助代码优化和重构建议
- 🧠 **记忆系统** - 记住你的编码习惯和项目信息
- 📝 **会话管理** - 按项目/文件管理对话历史

### 高级功能
- 📊 **项目分析** - 分析项目结构、技术栈、潜在问题
- 📄 **文件分析** - 深度分析当前文件的代码质量
- 📚 **文档生成** - 自动生成 README、API文档、架构文档等
- 💡 **解决方案** - 针对问题提供多种可行方案和实施步骤
- ✅ **代码审查** - 代码质量检查和改进建议（开发中）

### 即将推出
- 🚀 **流式响应** - 打字机效果的实时响应（等待 API 支持）
- 🤖 **智能补全** - 基于上下文的代码补全（等待 API 支持）
- 📦 **批量处理** - 多文件批量分析和处理（等待 API 支持）

## 🚀 快速开始

### 1. 配置 API Key

打开设置（`Ctrl+,`），搜索 "xiaoyu"，配置：
- `xiaoyu.apiKey`: 你的 API 密钥
- `xiaoyu.apiUrl`: API 服务器地址（默认：https://xiaoyu.api.ymd.cc）

### 2. 打开聊天面板

- 点击左侧活动栏的 **小玉AI** 图标
- 或使用命令面板（`Ctrl+Shift+P`）输入 "打开小玉聊天"

### 3. 使用代码功能

- 选中代码 → 右键 → "解释选中的代码"
- 选中代码 → 右键 → "重构选中的代码"
- 选中代码 → 右键 → "审查选中的代码"

### 4. 项目分析

- 命令面板 → "小玉AI: 分析整个项目"
- 命令面板 → "小玉AI: 生成项目文档"

## ⌨️ 快捷键

- `Ctrl+Enter`: 在聊天框中发送消息
- `Enter`: 换行
- `Shift+Enter`: 换行

## ⚙️ 配置项

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `xiaoyu.apiKey` | API 密钥 | "" |
| `xiaoyu.apiUrl` | API 服务器地址 | "https://xiaoyu.api.ymd.cc" |
| `xiaoyu.useMemory` | 启用记忆功能 | true |
| `xiaoyu.model` | 使用的模型 | "deepseek-chat" |

## 🎯 使用场景示例

### 场景 1：理解陌生代码
```
1. 打开项目文件
2. 选中不理解的代码段
3. 右键 → "解释选中的代码"
4. 在聊天面板查看详细解释
```

### 场景 2：项目文档生成
```
1. 打开项目
2. Ctrl+Shift+P → "小玉AI: 生成项目文档"
3. 选择文档类型（README、API文档等）
4. 自动生成专业文档
```

### 场景 3：解决技术问题
```
1. Ctrl+Shift+P → "小玉AI: 寻求解决方案"
2. 描述遇到的问题
3. 获取多种可行方案和实施步骤
```
